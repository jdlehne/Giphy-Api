<h2>Giphy-Api</h2>


Random gif generator with dynamically added user input buttons
Uses Giphy's REST API - User's can add their own search button, GIFs load in a paused state, they activate on click. Javascript does all the heavy lifting.

Visit at https://jdlehne.github.io/Giphy-Api/
